# Gallery Images

Add your photos here!

## Instructions:

1. Place your photos in this folder (`public/images/`)
2. Name them: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, etc.
3. Update the image array in `app/gallery/page.tsx` with:
   - Correct file names
   - Alt text descriptions
   - Optional captions for each photo

## Recommended Image Sizes:

- Minimum: 800x600px
- Recommended: 1200x900px or larger
- Format: JPG, PNG, or WebP

## Tips:

- Keep file sizes reasonable (under 2MB each)
- Use descriptive alt text for accessibility
- Add fun captions to make her smile!
